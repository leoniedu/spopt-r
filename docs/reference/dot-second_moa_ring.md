# Second areal moment for a single ring

Computes the second areal moment for a single ring

## Usage

``` r
.second_moa_ring(coords, centroid)
```

## Arguments

- coords:

  A matrix of coordinates for a ring

- centroid:

  Centroid coordinates, supplied as an object of class sf, sfc, or sfg

## Value

A single number, giving the second areal moment for a ring
