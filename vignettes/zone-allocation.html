Zone allocation with allocate_zones

allocate_zones() partitions geographic units into zones, each with a center, such that no unit exceeds max_distance from its center. It automatically discovers the minimum number of centers via LSCP, then assigns units using one of three methods:

  • "nearest": Greedy nearest-center assignment (fastest)
  • "p_median": Minimizes total weighted distance (optimal)
  • "cflp": Adds capacity constraints (balanced workloads)

This vignette demonstrates all three methods with TSP-based zone sequencing using Census tracts from Tarrant County, Texas.

Setup

library(spopt)
library(tidycensus)
library(tidyverse)
library(sf)

tarrant <- get_acs(
  geography = "tract",
  variables = "B01003_001",
  state = "TX",
  county = "Tarrant",
  geometry = TRUE,
  year = 2023
) |>
  filter(estimate > 0) |>
  rename(population = estimate)

zones_sf <- tarrant |>
  st_centroid() |>
  select(GEOID, population)

Three methods compared

result_near <- allocate_zones(zones_sf, max_distance = 5,
                              cost_matrix = dm, verbose = TRUE)

result_opt <- allocate_zones(zones_sf, max_distance = 5, method = "p_median",
                             weight_col = "population",
                             cost_matrix = dm, verbose = TRUE)

result_cap <- allocate_zones(zones_sf, max_distance = 5, method = "cflp",
                             capacity = 5,
                             cost_matrix = dm, verbose = TRUE)

Mapping

map_near <- tarrant |>
  left_join(st_drop_geometry(result_near$zones) |>
              select(GEOID, zone = .center), by = "GEOID") |>
  mutate(method = "nearest")

map_opt <- tarrant |>
  left_join(st_drop_geometry(result_opt$zones) |>
              select(GEOID, zone = .center), by = "GEOID") |>
  mutate(method = "p_median")

map_cap <- tarrant |>
  left_join(st_drop_geometry(result_cap$zones) |>
              select(GEOID, zone = .center), by = "GEOID") |>
  mutate(method = "cflp (cap=5)")

bind_rows(map_near, map_opt, map_cap) |>
  mutate(method = factor(method, levels = c("nearest", "p_median", "cflp (cap=5)"))) |>
  ggplot() +
  geom_sf(aes(fill = factor(zone)), color = "white", linewidth = 0.1) +
  scale_fill_discrete(guide = "none") +
  facet_wrap(~method, ncol = 3) +
  theme_void()

Zone sequencing

When sequence = TRUE, zones and tracts are ordered via TSP for geographic routing. Inter-zone distances use minimum pairwise distance between zone members (matching the surveyzones approach). Within each zone, tracts are oriented so the entry tract is closest to the previous zone’s exit.

result_seq <- allocate_zones(zones_sf, max_distance = 5, method = "p_median",
                             weight_col = "population",
                             cost_matrix = dm, sequence = TRUE, verbose = TRUE)

# Show first 15 tracts in visit order
